# MyApp

## Описание
MyApp — это приложение для управления пользователями и их заказами.

## Модели
- **User**: Модель пользователя с полями `first_name`, `last_name`, `email`, `phone_number`.
- **Order**: Модель заказа с полями `user`, `order_date`, `total_price`.

## Представления
- **UserListView**: Представление для отображения списка пользователей.
- **OrderDetailView**: Представление для отображения деталей заказа.

## URL-адреса
- `/users/`: Список пользователей.
- `/orders/<int:pk>/`: Детали заказа.

## Установка и запуск
Для установки и запуска приложения выполните следующие шаги:
1. Установите зависимости: `pip install -r requirements.txt`.
2. Примените миграции: `python manage.py migrate`.
3. Запустите сервер: `python manage.py runserver`.
