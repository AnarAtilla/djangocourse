# MyPro

## Описание
MyPro — это проект, состоящий из двух приложений: `library` и `lagerhouse`. Каждое приложение имеет свою специфическую функциональность и предназначено для управления различными аспектами бизнеса.

## Приложения

### Library
- **Описание**: Приложение для управления книгами, авторами и категориями.
- **Модели**: `Book`, `Author`, `Category`.
- **URL-адреса**: `/books/`, `/authors/`, `/categories/`.
- **Документация**: [Library README](library/README.md)

### Lagerhouse
- **Описание**: Приложение для управления складскими запасами, продуктами, заказами и клиентами.
- **Модели**: `Product`, `Order`, `OrderItem`.
- **URL-адреса**: `/products/`, `/orders/`.
- **Документация**: [Lagerhouse README](lagerhouse/README.md)

## Установка и запуск
Для установки и запуска проекта выполните следующие шаги:

1. **Клонируйте репозиторий**:
   ```bash
   git clone <URL_репозитория>
   cd MyPro
   
Создайте и активируйте виртуальное окружение:  
python -m venv venv
source venv/bin/activate  # Для Windows: venv\Scripts\activate
Установите зависимости:  
pip install -r requirements.txt
Примените миграции:  
python manage.py makemigrations
python manage.py migrate
Создайте суперпользователя:  
python manage.py createsuperuser
Запустите сервер:  
python manage.py runserver
Использование
После запуска сервера, вы можете получить доступ к приложениям по следующим URL-адресам:  
/library/: Приложение для управления книгами.
/lagerhouse/: Приложение для управления складом.
