# MyPro

## Описание
MyPro — это проект, состоящий из трех приложений: `library`, `lagerhouse` и `task_manager`. Каждое приложение имеет свою специфическую функциональность и предназначено для управления различными аспектами бизнеса.

## Приложения

### Library
- **Описание**: Приложение для управления книгами, авторами и категориями.
- **Модели**: `Book`, `Author`, `Category`.
- **URL-адреса**: `/books/`, `/authors/`, `/categories/`.
- **Документация**: [Library README](library/README.md)

### Lagerhouse
- **Описание**: Приложение для управления складскими запасами, продуктами, заказами и клиентами.
- **Модели**: `Product`, `Order`, `OrderItem`.
- **URL-адреса**: `/products/`, `/orders/`.
- **Документация**: [Lagerhouse README](lagerhouse/README.md)

### Task Manager
- **Описание**: Приложение для управления задачами, которое позволяет создавать, обновлять и отслеживать задачи и подзадачи.
- **Модели**: `Category`, `Task`, `SubTask`.
- **URL-адреса**: `/tasks/`, `/categories/`.
- **Документация**: [Task Manager README](task_manager/README.md)

## Установка и запуск
Для установки и запуска проекта выполните следующие шаги:

1. **Клонируйте репозиторий**:
   ```bash
   git clone <URL_репозитория>
   cd MyPro
   
   ## 1. Создайте и активируйте виртуальное окружение:
   python -m venv venv
source venv/bin/activate  # Для Windows: venv\Scripts\activate

## 2. Установите зависимости:

pip install -r requirements.txt

## 3. Примените миграции:

python manage.py makemigrations
python manage.py migrate

## 4. Создайте суперпользователя:
python manage.py createsuperuser

## 5. Запустите сервер:
python manage.py runserver

Использование
После запуска сервера, вы можете получить доступ к приложениям по следующим URL-адресам:

/library/: Приложение для управления книгами.

/lagerhouse/: Приложение для управления складом.

/tasks/: Приложение для управления задачами.
